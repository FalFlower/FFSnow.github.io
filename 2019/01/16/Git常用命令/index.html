Git常用命令

##配置身份（姓名和邮箱）

git config –global user.name “xxx”

git config –global user.email “xxx@xxx”

##查看身份

git config –global user.name

git config –global user,email

##创建代码仓库

git init

##查看本地所有git操作

ls -al

##提交/添加代码

git add xxx.xxxgit

add xxxgit add . //添加所有文件提交代码（全部添加完毕以后）

git commit -m “xxx”

##忽略文件

.gitignore//AS自动生成两个，一个在app目录下，一个在根目录下。可以在里边添加想要屏蔽的文件查看文件修改情况git status查看修改内容

git diffgit diff /xxx/xxxgit diff /xxx/xxx/xxx.java

##撤销未提交代码

git checkout app/src/main/java/com/example/dell/xxx/xxx.java

##取消添加代码

git reset HEAD app/src/main/java/com/example/dell/xxx/xxx.java

##查看提交记录

git loggit log cb4bfeda544c73257cd454966 9a0e794f4727832 -1(代表只想看一行记录)

git log cb4bfeda544c73257cd4549669a0e794f4727832 -1 -p(代表只想看具体修改了什么内容)

##查看当前版本库当中有哪些分支

git branch

##创建分支

git branch version1.0

##切换到分支

git checkout version1.0

##合并分支与主干

git merge version1.0

##删除分支

git branch -D version1.0

#####将代码下载到本地

git clone xxx

#####将本地内容同步到远程版本库上

git push origin master (origin 指定的时远程版本库的Git地址；master 指定的是同步到哪一个分支)

#####将远程版本库上的修改同步到本地，保存到一个origin/master分支上

git fetch origin mester

#####查看远程版本库修改的内容

get diff origin/master

#####合并到主分支

git merge origin/master

#####将远程版本库上获取最新的代码并且合并到本地

git pull origin master
